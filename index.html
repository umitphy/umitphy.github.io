Learning the English Language is fun

Since I started learning the English language, I have got a vision of a new world sight. It is kind of complicated to learn and apply to speak but in the long term, it is joy. Learning a new language is not a normal day activity in life. There are some weirdly named things. In your mother language, it was not named but in another language is called. This is something you cannot accept in the first step. It is weird but by the time you start searching these words by yourself and volubly. Then you have the idea if there are a lot of unnamed feelings in the English language, what about all languages in the world? What about the languages which were forgotten? Do not just learn a language. Try to feel it!
